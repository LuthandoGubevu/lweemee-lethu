# **App Name**: Lweemee Insights

## Core Features:

- TikTok Trend Tracking: Monitor trending topics and hashtags on TikTok in South Africa to identify emerging opportunities for content creation.
- Handle Mode Analytics: Track TikTok analytics (views, likes, shares, comments, and follower growth) for specific TikTok accounts.
- Manual Data Import: Allow users to import data for analysis when the TikTok API is unavailable or access is limited.
- Sentiment Analysis: Determine the sentiment (positive, negative, neutral) of comments on TikTok videos to understand audience reaction. LLM used as a tool.
- Customizable Reports: Generate clear and concise reports summarizing key TikTok metrics, trends, and insights.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) for trust and insights.
- Background color: Very light blue (#F0F5FF) for a clean and professional feel.
- Accent color: Bright purple (#A044FF) for highlights and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use minimalist and data-driven icons.
- Clean and structured layout with clear sections and information hierarchy.
